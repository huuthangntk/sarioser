# Global Typography Guidelines

## Font Families
- Primary: Vazirmatn (Persian/Farsi optimized)
- Secondary: Lalezar (for headings and emphasis)
- Fallback: system-ui, sans-serif

## Font Sizes
- Headings:
  - h1: 2.5rem (40px)
  - h2: 2rem (32px)
  - h3: 1.75rem (28px)
  - h4: 1.5rem (24px)
- Body: 1rem (16px)
- Small: 0.875rem (14px)

## Font Weights
- Light: 300
- Regular: 400
- Medium: 500
- Semibold: 600
- Bold: 700

## Line Heights
- Tight: 1.2
- Normal: 1.5
- Relaxed: 1.75

## Direction Support
- RTL (Persian/Farsi)
- LTR (English)
